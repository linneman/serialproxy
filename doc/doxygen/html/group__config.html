Detailed Description

Implementation of the configuration file parser. The following settings with their default attributes are currently supported:

proxy-crtl-address 127.0.0.1 # TCP/IP address to be served
proxy-crtl-port 6000 # TCP/IP port to listen at
server-ip-max-connections 10 # Maximum allowed IP connections
server-socket-filename /tmp/serialproxy.sock # Unix Domain Socket device filename
server-socket-max-connections 10 # Maximum allowed UDS connections
phy-filename /tmp/phy # physical (real) character device (proxy source) pty-symlink-name /tmp/symlink # symlink name to pseudo terminal (proxy dest)
